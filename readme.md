# 🕐 Clock 

## Overview

 This Clock is a deliberately frustrating and hilariously bad clock UI built as part of the **Creatively Bad UI Assignment**.

The component technically performs its job showing the current time but does so in the most annoying way possible. Users are greeted with insults, unnecessary interactions, and hidden features designed to make a simple task far more difficult than it should be.

---

## Component Chosen

**Clock / Time Display**

---

## Features

### Time Display

* Shows the current time when the user clicks the button.

### Random Insults

* Displays a random sarcastic message every time the user checks the time.

Examples:

* "Why are you checking the time again?"
* "Time is an illusion."
* "The clock is judging you."
* "Buy a watch."
* "Error 404: Motivation not found."

### Useless Exit Button

* The Exit button refuses to let the user leave and instead displays a message.

### Hidden 24-Hour Format

* The 24-hour format option is hidden initially and only appears after interacting with the clock.

### Excessive Styling

* Bright colors
* Glowing effects
* Unnecessary animations
* Poor visual hierarchy
